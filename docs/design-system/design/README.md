# Glint 设计稿（归档）

## 界面图（IDE 主界面）
- `draw-517ecd8b-*.png` / `draw-5c41cfef-*.png` — 主界面 · **选中代码**：⌥1 就近浮卡（徽标 `SELECTION`，来源「实时·调 AI 解释」，含「这是什么/逻辑怎么走/在上下文中的位置」三段 + 概念标签 + 底部动作）；右侧 ⌥4 架构鸟瞰 Treemap；底部 TRAJECTORY 轨迹条。
- `draw-a497ffca-*.png` — 主界面 · **文件焦点**：⌥1 浮卡（徽标 `FILE`，来源「预生成·库内直读」，单段「负责什么」）；顶部**维度切换条**（⌥1 为什么 / ⌥2 谁调用 / ⌥3 怎么执行 / ⌥4 在哪）；中文轨迹条（轨迹 · 我看过这些地方 · 清空 Esc）。
