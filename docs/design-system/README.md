# Glint Design System

Glint 设计系统文档目录。

- **`Glint-Design-System.md`** — 完整规范：设计原则、调色板（电蓝×黑灰，暗色优先）、字体（Inter + JetBrains Mono + 像素点阵字）、间距/圆角/描边/阴影/运动、Linear 风组件、图谱与节点（dagre/ELK 分层布局、实心节点）、可访问性、令牌总表。
- `design/`（占位）— 设计稿/原型，由使用者本人产出后归档于此。

调性：电蓝 + 黑灰、Pascal/终端字符的复古数字感，结构与组件向 Linear 看齐。后续补 `tokens.css` 与 `styleguide.html`，三者保持同步。所有文档随 PRD / Spec 一同纳入版本管理。
